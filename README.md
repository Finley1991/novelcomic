# NovelComic ✨

> 把小说变成视频，只需要一键。

---

## 🎬 这是什么？

NovelComic 是一个 **AI 创意工具箱**，专门为小说推文和视频混剪而生。

不用再一个个手动剪视频、配字幕、找素材 —— 把脏活累活都交给 AI，你只需要负责创意。

---

## 🚀 两大杀器

### 📖 小说漫剧生成
粘贴小说 → AI 自动提取角色场景 → 生成分镜 → 批量画图 → 配音 → 导出剪映草稿

**就这么简单。**

- 🎭 角色场景自动抓
- 🎨 提示词自己会优化（还看前后文！）
- 🖼️ ComfyUI 工作流随便换
- 🔊 24 种音色任你挑
- ✂️ 剪映 5.9+ 直接打开

### 🎥 解压视频混剪
写好推文 → 选个风格 → 自动选视频素材 → 生成风格图 → 一键导出多层轨道

**视频时长管够，字幕大小刚好。**

- 📹 素材库自动扫描
- ⏱️ 5 秒缓冲，再也不怕片尾被砍
- 🎨 风格提示词库持续更新（新加「水晶质感」✨）
- 🖼️ 每张图 15 秒，缩放动画安排上
- 📝 新青年体 13 号，手机看超舒服

---

## 💫 新增：剪映草稿微调

导出草稿后发现差点意思？直接在应用里改！

- 🎪 加个封面标题（格式帮你定好了，内容你随便写）
- 💬 加两行文本（位置格式都不用操心）
- 🏷️ 加水印（还能动！左上飘到右下那种）
- 🎵 加配乐（-27dB 响度帮你调好，循环裁剪自动来）

## 🛠️ 技术细节（给懂的人看）

| 部分 | 技术栈 |
|------|--------|
| 后端 | FastAPI + Pydantic + aiohttp |
| 前端 | React 18 + TypeScript + Tailwind + Vite |
| AI | ComfyUI + Ollama/OpenAI + Microsoft TTS |

**一些值得说的优化：**
- 随机种子真的随机了（之前工作流默认种子 0 会覆盖，已修）
- 视频选择会多算 5 秒，确保字幕播完
- 字幕字号从 15 改成 13，移动端观看体验 UP
- style_prompts 目录可以提交 git 了（projects 和 tmp 还是忽略的）

## 📦 快速上车

### 后端
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# 改改配置
python main.py
```

### 前端
```bash
cd frontend
npm install
npm run dev
```

然后打开 http://localhost:5173 开始玩！
