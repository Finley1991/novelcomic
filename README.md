## 功能特性

### 小说漫画推文系统

- **智能角色提取** - 基于大模型自动识别小说核心人物
- **场景深度理解** - 智能场景提取与关联，支持场景描述编辑
- **剧本分镜生成** - 支持 1/2/3 行智能分割，自动生成画图提示词
- **上下文增强提示** - 结合分镜、角色、场景及前后 5 镜上下文生成增强提示词
- **AI 图片生成** - ComfyUI 工作流支持，灵活的节点映射与参数配置
- **多音色配音** - Microsoft TTS 24 种音色，支持角色与分镜级独立配置
- **一键剪映导出** - pyJianYingDraft 驱动，兼容剪映 5.9+

### 解压视频混剪引擎

- **智能素材管理** - 自动扫描视频目录，实时获取视频时长
- **动态时长匹配** - 5 秒缓冲机制，确保视频时长完全覆盖音频
- **风格提示词库** - 多级目录结构，随机选择风格提示词
- **智能分段配音** - 按行自动拆分文本，批量 TTS 生成
- **AI 图片叠加** - 根据音频时长自动计算图片数量，每张 15 秒带随机动效
- **多层轨道导出** - 视频下层 + 图片上层 + 音频 + 字幕，9:16 竖屏配置
- **新青年体字幕** - 13 号字号优化，更适合移动端观看

### 剪映草稿调整工作台

- **实时草稿加载** - 直接读取剪映草稿 JSON 结构
- **封面标题系统** - 固定格式配置，支持自定义内容
- **文本水印动画** - 左上到右下关键帧动画，格式灵活配置
- **配乐响度标准化** - -27dB 响度自动匹配，智能循环裁剪
- **占位接口预留** - 封面图片上传、配乐导入功能框架完整

### 通用技术底座

- **多 LLM 提供商支持** - Ollama / OpenAI / 兼容 API 三选一
- **HTTP 代理配置** - 灵活的网络代理支持
- **现代化 UI/UX** - 温暖亲和配色系统，明暗双主题完整支持
- **向导式交互** - 侧边栏导航 + 向导式混合布局
- **键盘快捷键** - 高效操作体验
- **随机种子优化** - 真正的随机生成，确保每张图片独一无二

## 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+
- ComfyUI 服务
- LLM 服务（Ollama 或 OpenAI 兼容 API）
- Microsoft Azure TTS（可选）

### 后端启动

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# 编辑 .env 配置 API 地址
python main.py
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

### 访问地址

- 前端应用：http://localhost:5173
- 后端 API：http://localhost:8000
- API 文档：http://localhost:8000/docs

- HTTP 代理配置
- 连接测试验证

- Region 选择

- 草稿保存路径配置

## 工作流

### 小说漫画推文

1. **服务配置** - LLM、ComfyUI、TTS 一站式设置
2. **项目创建** - 小说文本导入，项目初始化
3. **场景管理** - 自动场景提取，支持手动编辑
4. **角色设计** - 智能角色识别，提示词与参考图管理
5. **剧本拆分** - 1/2/3 行灵活分割，自动关联场景角色
6. **提示词生成** - 上下文增强的 AI 画图提示词
7. **批量生图** - ComfyUI 工作流驱动，支持取消操作
8. **配音合成** - 多音色批量生成，支持试听
9. **剪映导出** - 一键生成专业草稿

### 解压视频混剪

1. **路径配置** - 视频素材库、风格提示词库设置
2. **文本导入** - 推文文本粘贴，项目创建
3. **风格选择** - 从风格库选择图片风格
4. **文本分段** - 按行自动拆分，生成配音片段
5. **智能选视频** - 根据总时长自动组合视频素材
6. **风格图生成** - 批量生成带缩放动画的风格图片
7. **草稿导出** - 多层轨道，9:16 竖屏配置

### 剪映草稿调整

1. **草稿加载** - 读取已生成的剪映草稿
2. **封面添加** - 标题格式固定，内容自定义
3. **文本叠加** - 位置格式固定，时长自动匹配
4. **水印动画** - 左上到右下关键帧移动
5. **配乐处理** - -27dB 响度标准化，智能循环裁剪
6. **草稿保存** - 直接写入剪映 JSON 结构

## 技术架构

```
novelcomic/
├── backend/
│   ├── api/              # FastAPI 路由层
│   ├── core/             # 核心业务逻辑
│   │   ├── comfyui.py    # ComfyUI 客户端
│   │   ├── decompression_exporter.py  # 剪映导出器
│   │   ├── decompression_utils.py     # 视频扫描与选择
│   │   └── draft_adjuster.py          # 草稿调整引擎
│   └── models/           # Pydantic 数据模型
├── frontend/
│   └── src/
│       ├── pages/        # React 页面组件
│       └── services/     # TypeScript API 客户端
└── data/
    ├── config.json       # 全局配置
    ├── style_prompts/    # 风格提示词库
    └── projects/         # 项目数据存储
```

### 核心技术栈

**后端**
- FastAPI - 现代异步 Web 框架
- Pydantic - 数据验证与序列化
- aiohttp - 异步 HTTP 客户端

**前端**
- React 18 - UI 框架
- TypeScript - 类型安全
- Tailwind CSS - 原子化样式
- Vite - 构建工具

**AI 服务**
- ComfyUI - 图片生成工作流
- Ollama / OpenAI - 大语言模型
- Microsoft TTS - 语音合成
