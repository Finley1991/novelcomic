# NovelComic 高端优化设计方案

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 优化 NovelComic 项目的视觉设计、交互体验和功能架构，参考 waoowaoo 项目的专业设计思路，打造高端大气的 AI 漫剧创作工具。

**Architecture:**
- 侧边栏导航 + 向导式项目编辑的混合布局
- 温暖亲和的视觉风格（浅色主题 + 深色主题切换）
- 专业化的功能模块（提示词工程、角色设定、分镜设计等）
- 流畅的交互体验（实时反馈、快捷操作、批量处理）

**Tech Stack:** React 18 + TypeScript + Tailwind CSS + FastAPI

## 一、整体架构设计

### 1.1 布局结构

**核心布局：侧边栏导航 + 向导式项目编辑**

```
┌─────────────────────────────────────────────────────────────────┐
│  NovelComic    [🔔 0]  [🌙/☀️]  [👤]                    │ ← 顶部栏
├──────────┬──────────────────────────────────────────────────────┤
│          │                                                      │
│  📁 项目 │              主内容区域                            │
│  🎨 模板 │                                                      │
│  ⚙️ 设置 │           (向导式布局 / 标签页式)                    │
│  📖 帮助 │                                                      │
│          │                                                      │
└──────────┴──────────────────────────────────────────────────────┘
```

**侧边栏功能：**
- 📁 **项目列表** - 快速切换项目，显示项目进度（进度条 + 完成度百分比）
- 🎨 **提示词模板** - 角色/场景/图像/分镜提示词管理
- ⚙️ **系统设置** - ComfyUI / LLM / TTS 配置
- 📖 **使用帮助** - 教程、快捷键、常见问题

**顶部栏功能：**
- Logo + 应用名称
- 通知中心（铃铛图标，显示未读数量）
- 主题切换（月亮/太阳图标，一键切换浅色/深色）
- 用户头像/设置入口

**向导式项目编辑：**
- 五步创作流程：项目设置 → 角色管理 → 场景管理 → 分镜编辑 → 导出交付
- 每个步骤有明确的完成状态标识（✅ 已完成 / ⚪ 进行中 / ⭕ 待处理）
- 顶部有"上一步"、"下一步"快捷按钮
- 侧边栏可折叠以获得更大编辑空间

### 1.2 页面路由结构

```
/                    - 仪表盘（项目列表）
/project/:id        - 项目编辑（向导式布局）
/templates          - 提示词模板管理器
/settings           - 系统设置
/help               - 帮助文档
```

## 三、交互体验设计

### 3.1 项目编辑 - 向导式布局

**五步创作流程：**

```
Step 1: 项目设置 → Step 2: 角色管理 → Step 3: 场景管理 → Step 4: 分镜编辑 → Step 5: 导出交付
   [✅ 完成]        [⚪ 进行中]      [⭕ 待处理]        [⭕ 待处理]        [⭕ 待处理]
```

#### 每个步骤的功能详情

**Step 1: 项目设置**
- 基本信息：项目名称、描述、创建时间
- 全局风格：风格提示词、否定提示词
- 小说文本：粘贴/编辑小说原文，支持大文本编辑
- 快捷操作：从模板创建项目、导入项目

**Step 2: 角色管理**
- 自动提取角色按钮
- 角色卡片网格布局
- 每个角色卡片包含：
  - 角色头像/参考图
  - 基本信息（姓名、性别、年龄）
  - 视觉描述摘要
  - 操作按钮（编辑、删除、复制）
- 角色详情弹窗：完整的角色设定编辑

**Step 3: 场景管理**
- 自动提取场景按钮
- 场景卡片列表布局
- 每个场景卡片包含：
  - 场景预览图（如有）
  - 场景名称和描述摘要
  - 操作按钮（编辑、删除、复制）
- 场景详情弹窗：完整的场景设定编辑

**Step 4: 分镜编辑**
- 剧本拆分功能（按 1/2/3 行）
- 分镜卡片时间轴布局
- 每个分镜卡片包含：
  - 分镜缩略图
  - 分镜序号和时长
  - 画面描述摘要
  - 音频状态指示器
  - 操作按钮（编辑、预览、删除）
- 批量操作工具栏
- 分镜详情侧边栏（点击分镜卡片展开）

**Step 5: 导出交付**
- 生成进度总览（图片/音频完成度）
- 剪映草稿导出功能
- 项目导出（ZIP 包含所有素材）
- 历史版本列表
- 分享功能（生成分享链接）

#### 步骤导航特性

- **已完成步骤**：显示 ✅ 图标，绿色高亮，点击可返回编辑
- **当前步骤**：显示 ⚪ 图标，橙色高亮，主内容区域
- **未完成步骤**：显示 ⭕ 图标，灰色，可点击跳转（但有提示确认）
- **顶部按钮**："上一步"、"下一步"，根据当前步骤状态启用/禁用
- **步骤提示**：每个步骤有智能提示，告诉用户该做什么

### 3.2 实时反馈设计

#### 加载状态

**骨架屏（Skeleton）：**
- 代替传统的 loading 转圈
- 卡片骨架：模拟卡片布局的灰色块
- 列表骨架：模拟列表项的灰色条
- 渐变动画：从左到右的扫光效果

**进度展示：**
- 进度条 + 百分比文字
- 当前操作说明（如"正在生成第 5/10 张图片..."）
- 预计剩余时间（如"预计还需 2 分 30 秒"）
- 取消按钮（如适用）

#### 操作反馈

**成功反馈：**
- 绿色小图标（✓）+ 简短提示文字
- 位置：操作按钮附近或右上角通知中心
- 自动消失：3 秒后自动淡出
- 可点击撤销：重要操作提供撤销选项

**失败反馈：**
- 红色小图标（✕）+ 错误信息
- 重试按钮：一键重新尝试
- 详细错误：展开查看技术细节
- 位置：操作按钮附近或右上角通知中心

**警告确认：**
- 黄色小图标（⚠️）+ 确认对话框
- 清晰的后果说明
- 明确的确认/取消按钮
- 不再提示选项：可记住用户选择

#### 批量操作

**选中状态：**
- 选中卡片有明显的边框和背景色
- 选中数量显示在工具栏
- 全选/取消全选按钮

**批量操作工具栏：**
- 固定在顶部/底部
- 包含：批量编辑、批量删除、批量生成等
- 操作进度实时展示
- 支持中途取消

## 四、功能架构设计（参考 waoowaoo）

### 4.1 提示词工程专业化

#### 提示词模板系统增强

```
提示词管理器
├── 角色提取模板
│   ├── 预设模板（4个）
│   │   ├── 通用版
│   │   ├── 动漫风格
│   │   ├── 写实风格
│   │   └── 古风/武侠
│   └── 自定义模板
├── 场景提取模板
│   ├── 预设模板（4个）
│   └── 自定义模板
├── 图像生成模板
│   ├── 预设模板（4个）
│   └── 自定义模板
└── 分镜提示词模板
    ├── 预设模板（4个）
    └── 自定义模板
```

#### 新功能详情

**模板版本管理：**
- 每个模板保存历史版本
- 版本列表显示：时间、修改人、变更摘要
- 一键回滚到任意历史版本
- 版本对比功能：高亮显示差异
- 自动版本保存：编辑时自动创建快照

**模板预览功能：**
- 测试输入区域：输入测试文本
- 实时预览输出：显示模板渲染结果
- 多版本对比：同时预览多个版本的输出
- JSON 格式化：美化显示输出结果
- 一键复制：复制预览结果到剪贴板

**模板分享功能：**
- 导出模板：导出为 JSON 文件
- 导入模板：从 JSON 文件导入
- 模板市场（远期）：分享和发现社区模板
- 模板打包：多个模板打包导出

**模板变量编辑器：**
- 可视化变量列表：显示所有可用变量
- 变量说明：每个变量的用途说明
- 变量插入：点击变量名插入到光标位置
- 自定义变量：支持创建项目特定变量
- 变量预览：显示变量替换后的效果

**A/B 测试功能：**
- 选择两个模板进行对比
- 相同输入，同时生成两个输出
- 并排对比视图
- 评分功能：为每个输出评分
- 历史对比记录：保存对比结果

│   ├── 核心性格（标签选择 + 自定义）

#### 新功能详情

**参考图智能分析：**
- 上传图片后自动提取视觉特征

- 自动填充到视觉描述字段
- 置信度显示：AI 对每个特征的置信度
- 人工确认：用户可修改自动提取的结果

**角色一致性检查：**
- 多张参考图风格一致性评分
- 不一致警告：指出可能的矛盾点
- 调整建议：提供一致性优化建议
- 风格统一工具：批量调整参考图风格

**角色表情库：**
- 预设常用表情（开心、生气、悲伤、惊讶、中性、思考等）
- 每个表情有示例提示词
- 表情快捷生成：一键生成该表情的角色图
- 自定义表情：用户可添加自定义表情

**角色 Sheet 导出：**
- 导出完整的角色设定文档（PDF/Markdown）
- 包含：所有角色信息、参考图、提示词
- 可自定义模板：支持自定义导出格式
- 批量导出：一次性导出所有角色
