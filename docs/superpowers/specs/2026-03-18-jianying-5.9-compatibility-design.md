
# 剪映 5.9 版本兼容性设计

**日期**: 2026-03-18
**项目**: NovelComic - 小说推文 AI漫剧生成工具

---

## 1. 概述

### 1.1 问题背景

当前剪映草稿生成器使用的是剪映 12.7.0 版本的格式（version: 400000, new_version: "127.0.0"），该格式与剪映 5.9 及之前版本不兼容。

### 1.2 解决方案

基于开源项目 [CutAuto](https://github.com/Hommy-master/CutAuto) 的验证模板，重构剪映草稿生成器，使其完全兼容剪映 5.9 版本。

### 1.3 参考来源

- **CutAuto 项目**: https://github.com/Hommy-master/CutAuto
- **关键模板文件**:
  - `src/pyJianYingDraft/assets/draft_content_template.json`
  - `src/pyJianYingDraft/assets/draft_meta_info.json`

---

## 2. 版本对比

### 2.1 关键版本号差异

| 字段 | 当前（新版） | 5.9 兼容版 |
|------|-------------|-----------|
| `version` | 400000 | 360000 |
| `new_version` | "127.0.0" | "110.0.0" |
| `platform.app_version` | "12.7.0" | "5.9.0" |
| `platform.app_id` | 5 | 3704 |
| `platform.app_source` | "official" | "lv" |

### 2.2 平台信息结构

**新版**:
```json
{
  "platform": {
    "os": "Windows",
    "os_version": "10.0.22631",
    "app_version": "12.7.0",
    "app_source": "official",
    "device_id": "",
    "hard_disk_id": "",
    "mac_address": "",
    "app_id": 5
  }
}
```

**5.9 兼容版**:
```json
{
  "platform": {
    "app_id": 3704,
    "app_source": "lv",
    "app_version": "5.9.0",
    "os": "windows"
  }
}
```

**关键差异**:
- 移除 `os_version`, `device_id`, `hard_disk_id`, `mac_address` 字段
- `os` 值从 "Windows" 改为 "windows"（小写）
- `app_id`: 5 → 3704
- `app_source`: "official" → "lv"
- `app_version`: "12.7.0" → "5.9.0"

### 2.3 其他字段差异

| 字段 | 当前（新版） | 5.9 兼容版 |
|------|-------------|-----------|
| `extra_info` | 复杂对象 | `null` |
| `color_space` | -1 | 0 |
| `render_index_track_mode_on` | `true` | `false` |

## 3. 架构设计

### 3.1 文件结构变更

```
novelcomic/backend/
├── core/
│   ├── jianying.py          # 重构后的主文件
│   └── assets/              # 新增：模板文件目录
│       ├── __init__.py
│       ├── draft_content_template.json
│       └── draft_meta_info.json
```

### 3.2 JianyingGenerator 类重构

**初始化流程**:
```
1. 加载 draft_content_template.json
2. 加载 draft_meta_info.json
3. 设置基础字段（id, name, canvas_config）
4. 生成素材
5. 生成轨道和片段
6. 保存 draft_content.json
7. 保存 draft_meta_info.json
8. 复制素材文件
9. 打包 ZIP
```
